# Customer Analytics Dashboard (Excel)

## Project Overview

This project is an interactive Customer Analytics Dashboard built in Microsoft Excel to analyze customer information using Pivot Tables, Pivot Charts, and Slicers.

The dashboard enables users to explore customer distribution across different states, cities, gender, customer segments, preferred channels, and customer status.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

---

## Dashboard Features

- Interactive Slicers
- Customer Distribution by State
- Customer Distribution by City
- Gender Analysis
- Preferred Channel Analysis
- Customer Segment Analysis
- Active vs Inactive Customer Analysis
